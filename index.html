1. HTML Boilerplate

    
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>

</body>
</html>
    Creates an HTML document

    2. BASIC TAGS

  2. Creates an HTML document
    <html> </html>

  3. Sets off the title & other info that isn't displayed
    <head> </head>

  4. Sets off the visible portion of the document
    <body> </body>

  5. Puts name of the document in the title bar; when bookmarking pages, this is what is bookmarked
    <title> </title>

    6. TEXT TAGS

  6. Creates preformatted text
    <pre> </pre>

  7. Creates headlines -- H1=largest, H6=smallest
    <h1> </h1> ... <h6> </h6>

  8. Creates bold text (should use <strong> instead)
    <b> </b>

  9. Creates italicized text (should use <em> instead)
    <i> </i>

  10. Creates typewriter-style text
    <tt> </tt>

  11. Used to define source code, usually monospace
    <code> </code>

  12. Creates a citation, usually processed in italics
    <cite> </cite>
